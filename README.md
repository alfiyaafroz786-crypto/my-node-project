# Assignment 6 - Simple Web Server using NodeJS

## Route
1. /home or / = Home Page
2. /about = Avout Us Page
3. /service = Service Page
4. /contact = Contact us Page
5. /anyPath = 404 Not found

## Project Setup command:
    > npm init
    > npm i nodemon --save-dev
    > npm run dev
